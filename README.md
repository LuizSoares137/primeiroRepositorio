# Como criar um repositorio no GitHub
é nescessario ter instalado o **GIt** no computador.
Vamos usar tbm o **Github desktop** para facilitar.

## passo a passo
supondo que vc ja tem um projeto criado, faça o seguinte:
- Abrir o CMD dentro da pasta e digitar o comando ``git init``
- ir Github no desktop e escolher a opçao ``Add existing repository`` e selecionar a pasta do projeto.
- Por fim, fazer o commit e publicar, **Atencao:** lembrar de deixar publico.
